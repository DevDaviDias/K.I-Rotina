Widget iOS

Como adicionar a Rotina à tela inicial como widget usando Atalhos

1

Instale o app na tela inicial

Abra o app Rotina no Safari → toque em Compartilhar → Adicionar à Tela de Início. Isso é obrigatório para o widget funcionar.

2

Abra o app Atalhos

No seu iPhone, abra o app Atalhos (pré-instalado). Toque em + para criar um novo atalho.

3

Adicione a ação "Obter conteúdo de URL"

Pesquise por Obter conteúdo de URL e adicione a ação. Na URL, insira o endereço do seu app Rotina (ex: https://seuapp.com).

Isso vai carregar a página e permitir extrair dados do localStorage via JavaScript.

4

Use "Executar JavaScript no Safari" (alternativa mais simples)

Adicione a ação Abrir URLs apontando para o app. Para um widget de texto simples, use o atalho de Scriptable (app gratuito) que lê o localStorage:

Script para Scriptable (widget)

// Cole no app Scriptable como widget
const url = "https://seuapp.com"
const req = new Request(url)
const html = await req.loadString()

// Extrai dados do rotina_widget salvo no localStorage
// (requer que o app esteja aberto ao menos uma vez)
const wm = importModule("no-background")
const widget = new ListWidget()
widget.backgroundColor = new Color("#0a0a0a")

const title = widget.addText("Rotina")
title.textColor = new Color("#c8f065")
title.font = Font.boldMonospacedSystemFont(12)

widget.addSpacer(4)
const sub = widget.addText("Abra o app para ver")
sub.textColor = new Color("#555555")
sub.font = Font.monospacedSystemFont(11)

Script.setWidget(widget)
widget.presentSmall()

Opção mais simples: Widget de atalho

1. Crie um atalho que abre o app Rotina
2. Na tela inicial, segure → Editar → +
3. Procure "Atalhos" na lista de widgets
4. Escolha o atalho criado
5. Isso cria um botão que abre o app direto na tela atual

💡 O app Rotina já expõe os dados da tarefa atual em localStorage["rotina_widget"] como JSON. Apps como Scriptable conseguem ler isso via WebView para montar widgets nativos.
